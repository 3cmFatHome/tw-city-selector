安裝

擇一使用下列方式

- 透過 Bower

bower install tw-city-selector --save

- 透過 Yarn

yarn add tw-city-selector

- 透過 NPM

npm install tw-city-selector --save

- 下載檔案

Github

引入

無依賴，直接引入至頁面中 (記得將路徑改為實際檔案路徑)

<script src="your/path/tw-city-selector.min.js"></script>

使用方式

可使用下列兩種方式加載選單元件：

  • element 加上 role-attribute (預設用法，自訂性低)
  • js 實例 (自訂性高)

以下分別說明

element 加上 role-attribute

設置一個空 element，加入 role-attribute

<div role="tw-city-selector"></div>

* 產生的選單組合當中，預設包含一個隱藏的郵遞區號欄位

<input class="zipcode" name="zipcode" type="text" readonly="" autocomplete="off" placeholder="郵遞區號" style="width: 6em; display: none;">

顯示郵遞區號欄位

<div role="tw-city-selector" data-show-zipcode></div>

限制部分縣市 (例：臺北市、臺中市、高雄市)

<div role="tw-city-selector" data-only="臺北市,臺中市,高雄市"></div>

預設選定 (例：新北市及其板橋區)

<div role="tw-city-selector"
     data-selected-country="新北市"
     data-selected-district="板橋區">
</div>

限制部分縣市 & 預設選定

<div role="tw-city-selector"
     data-only="臺北市,臺中市,高雄市"
     data-selected-country="臺中市"
     data-selected-district="北區">
</div>

調整樣式

在 tag 加上你想要的 class name，好讓他吃到 stylesheet 的設定

<div role="tw-city-selector" class="my-selector"></div>

<style>
.my-selector select {
 -webkit-appearance: none;
 -moz-appearance: none;
 appearance: none;
 outline: none;
 border-color: #ffe76f;
 border-width: 3px;
 color: #ff8e52;
}
</style>
* 上述樣式調整僅為示意，非本頁面選單樣式設定

套用 Bootstrap 的選單 class name

<div role="tw-city-selector" data-bootstrap-style></div>
