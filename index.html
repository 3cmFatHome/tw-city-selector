tw-city-selector.js

台灣縣市選單，真是 nice

安裝

擇一使用下列方式

透過 Bower

bower install tw-city-selector

下載檔案

至 Github 下載

引入 js

無其他依賴，將下列路徑改為套件位置

<script src="tw-city-selector.min.js"></script>

使用方式

可使用兩種方式做元件加載：

  • tag 加上 role-attribute
  • javascript 呼叫實例

tag 加上 role-attribute

預設用法 (附有隱藏的郵遞區號欄位)
<div role="tw-city-selector"></div>
顯示郵遞區號欄位
<div role="tw-city-selector" data-show-zipcode></div>
限制可選擇縣市 (例：台北市、台中市、高雄市)
<div role="tw-city-selector" data-only="台北市,台中市,高雄市"></div>
選定縣市 (例：新北市及其板橋區)
<div role="tw-city-selector"
     data-selected-country="新北市"
     data-selected-district="板橋區">
</div>
限制可選擇縣市 & 選定縣市
<div role="tw-city-selector"
     data-only="台北市,台中市,高雄市"
     data-selected-country="台中市"
     data-selected-district="北區">
</div>
全部縣市
設定顯示部分縣市-台北市、台中市、高雄市
選定新北市及其板橋區
設定顯示部分縣市 & 選定台中市及其北區
使用已存在的 dom (例：Bootstrap 樣式)
